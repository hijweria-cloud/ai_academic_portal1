# AI-Based Academic Portal & Learning Management System
Developed for GC University Faisalabad (GCUF).

## Features
- AI-Powered GPA Prediction (Decision Tree Regressor)
- Real-time Academic Performance Tracking
- Student Verification System

## Tech Stack
- **Frontend:** Bootstrap 5, Poppins Font
- **Backend:** PHP 8.x
- **Database:** MySQL
- **AI Engine:** Python 3.x (Scikit-learn)

## Installation
1. Import `sql/ai_academic_portal.sql` into phpMyAdmin.
2. Update `includes/db_connect.php` with your database credentials.
3. Run `pip install -r requirements.txt` for AI modules.
